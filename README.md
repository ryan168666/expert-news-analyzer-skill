# Expert News Analyzer Skill 📊

OpenClaw 技能版 - 专业的 AI 驱动情报降噪与决策辅助系统。

🌟 核心价值：对抗信息熵增
在碎片化信息泛滥的时代，本工具通过 物理降噪 与 逻辑对齐，将海量原始数据转化为具备决策深度的结构化情报。

1. 多智能体专家矩阵 (Multi-Agent Matrix)
- 角色驱动分析：模拟巴菲特（价值投资）、马斯克（第一性原理）、达利欧（风险平价）等 11 位投资大师的独立思维模型。
- 逻辑对齐：大模型根据预设的 persona 和 expertise，自动将新闻变量注入专家的决策框架。
- 圆桌讨论逻辑：通过 LLM 汇总各方分歧，消除单一代理的认知偏差，输出高置信度的降噪结论。

2. 逻辑正义的 RAG 架构 (Robust RAG)
- 时效性感知：引入 180 天半衰期衰减函数（Half-life Decay），解决金融领域"过期信息误导"的痛点。
- 同义词防稀释：区分原词（权重 1.0）与同义词（权重 0.5），并以原词数作为归一化分母，保护检索精准度。
- 上下文完整性：实现 50 字符分块重叠（Chunk Overlap），防止专家逻辑链条被物理切断。

3. 智能双轨过滤 (Dual-Scoring System)
- 客观评分：基于关键词权重（如"美联储"、"AI"等）与来源权威度（如 Bloomberg, Reuters）进行初步物理降噪。
- AI 价值评估：利用 LLM 从影响力、时效性、可操作性等 5 个维度对新闻进行 0-100 的深度评分。

## 功能

- 📰 多源新闻聚合
- 🤖 AI 专家分析（5-10位专家）
- 📈 趋势信号
- 📱 微信推送
- ⏰ 定时自动运行

## 快速开始

```bash
# 安装技能
openclaw skill install expert-news-analyzer-skill

# 运行
python3 analyzer_v2.2.py
```
